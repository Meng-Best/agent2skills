## 格式说明

### 标题层级

```
# 一级标题（论文标题）
## 二级标题（章节：Introduction, Method 等）
### 三级标题（子章节：3.1, 3.2 等）
#### 四级标题（更细分的小节）
```

### 引用格式（英文原文）

使用 `>` 标记英文原文，与中文译文形成对照：

```markdown
> This is the original English text.

这是对应的中文翻译。
```

### 强调格式

```markdown
**加粗** - 用于重点内容、关键结论
*斜体* - 用于术语首次出现时的英文原文（可选）
`代码` - 用于变量名、函数名、命令等
```

### 表格格式

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

### 公式格式

行内公式：`$E = mc^2$`

行间公式：
```markdown
$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$
```

### 图表引用

```markdown
如图 1 所示，... (参见原文 Figure 1)

如表 2 所示，... (参见原文 Table 2)
```

## 文件命名规则

| 来源类型 | 输入示例 | 输出文件名 |
|----------|----------|------------|
| 本地 PDF | `D:\papers\attention.pdf` | `attention_翻译.md` |
| arXiv ID | `arxiv:1706.03762` | `1706.03762_翻译.md` |
| arXiv URL | `https://arxiv.org/abs/2301.00001` | `2301.00001_翻译.md` |
| DOI | `doi:10.1038/xxx` | `10.1038_xxx_翻译.md`（替换 `/` 为 `_`）|
